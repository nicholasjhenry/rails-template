h1. Template Description

* Runs with mysql because it uses the query_reviewer plugin.
* Assumes a Heroku deployment platform
* Uses Bundler for Gem management

h1. Setup/Installation

$ rails myproject -d mysql -m firsthand.rb

h1. Next Steps

Then give it a whirl:

$ cd myproject
$ rake db:create
$ script/generate scaffold post title:string body:text
$ rake db:migrate
$ script/server

Visit:

http://localhost:3000/posts
